rwt_keyboard_teleop
====================

Usage
-----
```
roslaunch navros_pkg cafe_custom.launch
rosrun web_video_server web_video_server 
roslaunch rosbridge_server rosbridge_websocket.launch

```
- Open file "index.html"
- Control using Joystick
- Add camera topic
- Press Load video button
